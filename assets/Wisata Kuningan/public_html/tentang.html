Tentang Aplikasi dan Developer

Aplikasi "Obyek Wisata Kuningan" merupakan aplikasi yang berfungsi untuk memberikan informasi mengenai lokasi tempat wisata di Kabupaten Kuningan, Jawa Barat. Aplikasi ini dikembangkan dengan menggunakan Google Maps API v.3, Twitter Bootstrap API, CodeIgniter Framework dan Laurens Framework dari 3Gen-ITDev. Aplikasi "Obyek Wisata Kuningan" dapat diterapkan pada perangkat smartphone Android 2.3.3 (Gingerbread) sampai dengan versi terbaru.
